# 信息安全数学教程讲义 (Mathematical Lecture Notes for Information Security)

本仓库是信息安全/密码学方向的核心数学基础讲义，覆盖离散代数、有限域、椭圆曲线、后量子密码四大核心板块，内容以 Beamer 幻灯片形式组织，兼顾理论严谨性与工业级实践指导，适配经典密码、工程实现、后量子密码三大学习路径。

## 📚 核心内容目录

### 第一部分：离散代数结构（经典密码的根基）

#### 1. 数论基础（安全参数的数学源头）
- 同余与模运算：密码算法的有限域运算基础
- 整除、互质、GCD/LCM 定义与高效计算（辗转相除法/扩展欧几里得）
- 贝祖定理：模逆元存在的理论基础（RSA 密钥生成核心）
- 算术基本定理与素数筛选：安全素数/大素数生成的底层逻辑

#### 2. 群论基础（密码系统的代数基石）
- 群的四大公理（封闭性/结合律/单位元/逆元）及密码学约束
- 阿贝尔群（交换群）与非交换群的密码学应用场景
- 子群、正规子群、商群：群的结构分解与工程价值
- 群同态/同构：代数结构的等价性与密码运算化简

#### 3. 环论与理想（现代密码的结构支撑）
- 环的公理化定义：加法阿贝尔群 + 乘法封闭 + 分配律
- 模 m 剩余类环 $\mathbb{Z}_m$：零因子、整环、域的密码安全边界
- 理想/主理想/极大理想：有限域构造的核心工具
- 商环与有限域：AES/SM4 中 $\text{GF}(2^8)$ 的工程实现

#### 4. 原根与离散对数（公钥密码的核心难题）
- 有限循环群的同构性（与 $\mathbb{Z}_n$ 同构）
- 生成元（原根）求解与安全素数选择
- 离散对数问题（DLP）：公钥密码的核心难问题
- 工业级实践：安全素数验证、大整数运算库（GMP/OpenSSL）

### 第二部分：有限域与结构构造（现代密码的工程基础）

#### 5. 有限域基础
- 素域 $\mathbb{F}_p$ 与扩域 $\mathbb{F}_{p^n}$ 的构造
- 不可约多项式选择：AES $\text{GF}(2^8)$ 的多项式环理想构造
- 有限域运算：加法/乘法/逆元的工程实现（字节级优化）
- 域的扩张：从 $\mathbb{Z}_2$ 到 $\text{GF}(2^8)$ 的工业级映射

#### 6. 有限域的密码应用
- AES/SM4 字节代换（S盒）的有限域逆元实现
- 列混合运算的多项式乘法优化
- 有限域在纠错码（CRC/ECC）、哈希函数中的应用
- 有限域运算的硬件/软件适配（指令集优化）

### 第三部分：椭圆曲线密码（经典公钥的高效形态）

#### 7. 椭圆曲线的代数基础
- 椭圆曲线的群结构定义（点加/点乘）
- 有限域上的椭圆曲线：$\text{GF}(p)$/$\text{GF}(2^n)$ 上的曲线方程
- 椭圆曲线离散对数问题（ECDLP）：抗攻击特性与复杂度分析

#### 8. ECC/SM2 工程实现
- SM2 椭圆曲线参数选择规范（国密标准）
- 点运算优化：仿射坐标/射影坐标转换
- 抗侧信道攻击的椭圆曲线运算实现
- ECC 与 RSA 对比：效率与安全性权衡

### 第四部分：量子威胁与后量子密码

#### 9. 经典密码的量子风险
- Shor 算法：大数分解/离散对数的量子求解
- ECC/RSA 算法的量子安全边界
- 后量子密码的设计目标：抗量子计算、兼容现有系统

#### 10. 格与后量子密码
- 格的基本定义：基、体积、正交化
- 格的核心难问题：SVP（最短向量问题）/CVP（最近向量问题）
- 学习同态加密（LWE/RLWE）：后量子密码的核心难题
- 后量子密码标准：Kyber（密钥封装）/Dilithium（数字签名）

### 第五部分：工程应用与避坑指南
- RSA/DH/SM2 算法的代数底层逻辑
- 椭圆曲线密码的子群运算优化
- 商群/理想在匿名密码、哈希函数中的应用
- 小阶子群攻击抵御、大整数运算溢出规避
- 后量子密码的工程适配（性能/兼容性权衡）

## 🎯 核心特色
1. **多路径适配**：支持经典密码、工程实现、后量子密码三条学习路径，按需选择  
2. **理论+工程**：每个数学定义都配套密码学应用场景，避免纯理论脱离实践  
3. **实例驱动**：所有定理均附计算实例（如模7乘法群生成元、AES S盒构造、SM2点乘）  
4. **可视化讲解**：包含 TikZ 绘制的几何直观图（如 GCD 矩形铺满、陪集划分、格的向量空间）  
5. **工业级标准**：对标 TLS/SSH/SM2/后量子密码标准，给出参数选择规范  

## 📋 使用说明

### 1. 环境依赖
- LaTeX 编译器：支持 Beamer 的 PDFLaTeX/XeLaTeX（如 TeX Live、MikTeX）  
- 必备宏包：`beamer`、`tikz`、`amsmath`、`amssymb`、`array`、`tabularx`、`pgfplots`  
- 大整数运算验证：可使用 GMP 库（C/C++）、OpenSSL BIGNUM 接口或 Python `cryptography` 库  
- 后量子密码验证：可使用 liboqs 库（Open Quantum Safe）  

### 2. 学习路径建议
- **经典密码路径**：同余 → 群 → 原根 → DLP → ECC  
- **工程实现路径**：有限域 → 域扩张 → 椭圆曲线 → SM2  
- **后量子密码路径**：线性代数 → 格 → SVP/CVP → LWE/RLWE
