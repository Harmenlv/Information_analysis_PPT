# 信息安全数学教程讲义 (Mathematical Lecture Notes for Information Security)
本仓库是信息安全/密码学方向的核心数学基础讲义，聚焦群论、数论及其在密码算法中的工程应用，内容以 Beamer 幻灯片形式组织，兼顾理论严谨性与工业级实践指导。

## 📚 核心内容目录
### 1. 群论基础（密码系统的代数基石）
- 群的四大公理（封闭性/结合律/单位元/逆元）及密码学约束
- 阿贝尔群（交换群）与非交换群的密码学应用场景
- 子群、正规子群、商群：群的结构分解与工程价值
- 群同态/同构：代数结构的等价性与密码运算化简

### 2. 数论核心（安全参数的数学源头）
- 整除、互质、GCD/LCM 定义与高效计算（辗转相除法）
- 贝祖定理：模逆元存在的理论基础（RSA 密钥生成核心）
- 算术基本定理（唯一分解定理）与素数筛选（埃拉托斯特尼筛法）
- 有限域 $\mathbb{F}_p$ 与乘法逆元求解（扩展欧几里得算法）

### 3. 循环群（公钥密码的核心载体）
- 有限循环群的同构性（与 $\mathbb{Z}_n$ 同构）
- 生成元（原根）求解与安全素数选择
- 公钥密码统一范式：大阶循环群 + 易算难逆的指数映射
- 工业级实践：安全素数、生成元验证、大整数运算库（GMP/OpenSSL）

### 4. 工程应用与避坑指南
- RSA/DH/SM2 算法的群论底层逻辑
- 椭圆曲线密码（ECC）的子群运算优化
- 商群在匿名密码（环签名）、哈希函数（SHA-256）中的应用
- 小阶子群攻击抵御、大整数运算溢出规避

## 🎯 核心特色
1. **理论+工程**：每个数学定义都配套密码学应用场景，避免纯理论脱离实践
2. **实例驱动**：所有定理均附计算实例（如模7乘法群生成元、RSA 小素数测试）
3. **可视化讲解**：包含 TikZ 绘制的几何直观图（如 GCD 矩形铺满、陪集划分）
4. **工业级标准**：对标 TLS/SSH/SM2 等商用密码标准，给出参数选择规范
